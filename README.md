# PDF AI Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using:

- LangChain
- Hugging Face Embeddings
- ChromaDB
- Mistral AI
- Streamlit

## Features

- Upload PDF
- Semantic Search
- Conversational Chat
- MMR Retrieval
- Chroma Vector Database

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py
